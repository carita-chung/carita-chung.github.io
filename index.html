About Me

Recipely

Making recipes accessible for all.

A web application for that gives replacements for ingredients for those who are vegetarians, vegans, halal, lactose-intolerant, gluten-free, and kosher! Recipely is a web app that helps individuals try new recipes despite their dietary needs.

This project was submitted to Hack The North 2020++.

replACE it!

A chrome extension that replaces non-inclusive language with inclusive alternatives.

Our team of four female students began chatting about our common shared struggles in our professional lives. As university students who are all beginning our professional careers, we discussed how common it is for women to say things like “I just…”, “I hope that’s ok”, “no worries if not”, and saying “sorry” many times when an apology is definitely unnecessary.

We built a Chrome Extension that replaces certain non-assertive phrases for assertive phrases. This project was submitted to SheHacks V.

Kitchen Timer

A simple Arduino Uno Timer.

The Arduino UNO is a microcontroller development board based on a 8 bit AVR mircocontroller. The development platform used is the Arduino IDE and there are a large number of open source libraries available, this makes it idea to experiment with common electronic components and to create simple electronic projects quickly.

WEMOS-Based Bus Schedule Display

Displays a realtime bus schedule.

The WEMOS is a relatively new mini controller board/module that has WIFI capability. It is also a board that is supported by the Arduino development system. This makes it an ideal system for IOT exploration. A bus schedule checker is a very useful application as a lot of people rely on bus for transportation on a daily basis.
